# Lakshya Shorthand Typing Institute (LIST) Website

A premium, responsive website for Lakshya Shorthand Typing Institute (LIST) in Faridabad.

## Features
- **Modern Design**: Built with glassmorphism, smooth gradients, and DM Sans/Playfair Display typography.
- **Interactive UI**: Includes a typewriter effect for exam targets and animated counters for student statistics.
- **Fully Responsive**: Optimized for all devices—mobile, tablet, and desktop.
- **SEO Optimized**: Includes meta tags and semantic HTML for better search engine visibility.
- **Interactive Elements**:
  - Smooth scroll animations.
  - Hamburger menu for mobile.
  - Testimonial slider.
  - Integrated Google Maps and WhatsApp contact.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with variables and animations.
- **JavaScript**: Vanilla JS for interactive features.

## How to Run
Simply open `index.html` in any modern web browser or serve it using a local server (like Live Server in VS Code).
